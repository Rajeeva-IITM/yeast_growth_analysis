# Genotype Analysis

Chromosome-wise genotype structure and mutation correlation analysis.

## Files

### `genotype_analysis.R`
Computes and visualizes correlation matrices of mutations within each of the 16 yeast chromosomes. Uses Bloom2013 genotype data, filtering to varying loci only. Outputs per-chromosome correlation heatmaps to visualize linkage disequilibrium patterns.

## Dependencies

- `src/utils/theme_set.R`
