# R Analysis - Yeast Growth Prediction

R-based downstream analysis for the yeast growth prediction project. This sub-repo handles statistical analysis, visualization, and biological interpretation of results generated by the companion Python (ML models, SHAP values) and MATLAB (metabolic modeling) pipelines.

## Setup

- **R version**: 4.3+
- **Dependency management**: [renv](https://rstudio.github.io/renv/)
- **IDE**: RStudio (project file: `R_analysis-yeast_growth_pred.Rproj`)

```r
# Restore the R environment
renv::restore()
```

All scripts assume the working directory is the project root.

## Repository Structure

```
.
├── src/                   # Analysis scripts (see subdirectory READMEs)
│   ├── Contingency/       # Statistical association tests
│   ├── data_exploration/  # Data QA and exploratory analysis
│   ├── expression/        # Differential gene expression
│   ├── genotype/          # Genotype structure analysis
│   ├── grn/               # Gene regulatory networks
│   ├── gsmm/              # Genome-scale metabolic model analysis
│   ├── shap/              # SHAP feature importance and latent interpretation
│   ├── utils/             # Shared utility functions
│   └── result_model_evaluation.qmd  # Cross-dataset validation report (Quarto)
├── data/                  # Input data (training data, predictions, SHAP values, etc.)
├── results/               # Generated figures and tables
├── renv/                  # R environment (managed by renv)
└── renv.lock              # Dependency lockfile
```

## External Dependencies

This repo expects pre-computed outputs from sibling repositories:

- **SHAP values** (CSV) from the Python ML pipeline
- **Model predictions** (CSV/Parquet) from classification and regression models
- **Flux sampling results** from the MATLAB metabolic modeling pipeline
- **QTL data** from external genetic mapping analyses
